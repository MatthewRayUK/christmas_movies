About This Project

We all love Christmas, don’t we? In my family, my sister and I are particularly obsessed. So, we wanted a fun way to track the movies we want to watch during December—ideally one each day as we count down to Christmas. As I learned more about APIs and Flask, I started brainstorming how to structure this idea and began putting together a few pages. Below, I’ll outline the development of this project and will continue to update it as I make progress.

  • Choosing Libraries: I began by selecting the libraries that would support me in Python. Flask is the main framework I’m using, alongside SQLAlchemy for database management, Flask-Bootstrap to help with HTML & CSS styling, and Flask-WTF to make handling forms a lot easier.
  • Setting Up Databases & Pages: The setup of the database and basic pages/navigation was straightforward, though I had to dive into Jinja (Flask’s templating engine) to embed Python code within my HTML documents.
  • API Integration: TheMovieDB API was the perfect resource for retrieving movie information. I set up an account and integrated it to allow users to search for films and fetch their details for the watchlist.
  • User Management: One of the most fun and educational parts of this project was learning about password security. I loved diving into hashing and salting passwords to keep user data safe, and I’ve implemented this securely within the app.
